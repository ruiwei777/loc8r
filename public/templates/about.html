About

This app fetches a location list within a specific range (say, 10km) using REST API from mlab. Then the result will be displayed by Angular.js. A search filtering the content is supported.

It is a mockup app following this book, but completely changed into a component-based AngularJS single page application.

Technology Specification

API Google Static Maps, mlab RESTful MongoDB
Technology Stack MongoDB, Express, Angular.js 1.5, Node.js 8, jQuery, Bootstrap
Deployed On Heroku

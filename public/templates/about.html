About

This app fetches a list of locations through REST API from mlab. Then the result will be rendered by Angular.js. A search filtering the content is supported.

Even though today Angular.js is no longer the trend, this app is to familiarise myself with typical usage of Angular.js in case I need to maintain old projects. It uses ui-router, components, $resource and services for Angular.js and Gulp & Bower for build tool & package management. Feel free to review the code structure.

Technology Stack MongoDB, Express, Angular.js, Node.js 8, Bootstrap

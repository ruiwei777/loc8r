About

This app fetches a list of locations through REST API from mlab. Then the result will be rendered by Angular.js. A search filtering the content is supported.

Some Angular.js modules used:
  • ui-router
  • components
  • $resource
  • ngAnimate
Build tools & package management:
  • Gulp
  • Bower

Technology Specification

Technology Stack MongoDB, Express, Angular.js, Node.js 8, Bootstrap
Deployed On Heroku
